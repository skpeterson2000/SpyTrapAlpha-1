# Alert sounds

Drop your audio files here and map them to alert tiers in `config.local.json`:

```json
{
  "audio": {
    "sounds": {
      "HIGH": "my-urgent.mp3",
      "MED":  "my-warning.wav",
      "_default": "my-beep.wav"
    }
  }
}
```

Supported out of the box: `.wav`, `.mp3`, `.ogg` (the player is auto-detected ‚Äî
`paplay`/`aplay`/`mpg123`/`ffplay`/`cvlc`). The engine plays **one** sound per
detection pass (the most severe alert), so longer "tracks" are fine.

The committed `alert-high.wav` / `alert-med.wav` / `alert.wav` are placeholder
tones generated by `tools/gen_test_sounds.py`; replace them with your own.
